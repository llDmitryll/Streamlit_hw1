Что было сделано
1. Удаление дубликатов, заполнение пропусков, обработка значений приведение к типам.
2. Анализ данных, псчитали основные статистики.
3. Визуализация попарных распределений, а ходе анализа этих графиков были выдвинуты гипотезы о зависчимостях между ними.
4. Таблица корреляций и её анализ.
5. Обучили модели на вещественных признаках: Elastic Net, Lasso, classic linear regression
6. Добавили категориальные фичи и обучили модель Ridge регрессии.
7. В ходе всех экспериментов с моделями анализировали изменения таких метрик MSE и R2score, а также оценили модели по своей метрике.
По качеству все модели показали себя примерно одинаково, огромного прироста нигде не наблюдается. С кросс валидацией чуть-чуть получше.

Был разработан сервис на streamlit, пока что можно загрузить предобработанные данные и посмотреть на основные графики, веса обученной модели и основные метрики.
Можно добавить предобработку и возможность пользователю вручную ввести свои данные.
